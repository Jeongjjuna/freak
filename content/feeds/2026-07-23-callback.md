## Callback — "끝나면 다시 알려줄게"

Callback은 **내가 요청한 작업이 끝났을 때, 다시 알려주는 방식**입니다.

### 포인트1 — 내가 먼저 요청한다

```java
paymentService.pay(request, result -> {
    System.out.println("결제 완료: " + result);
});
```

작업을 요청하면서 "끝나면 알려줘."라고 약속하는 것입니다.

프로그래밍에서는 **함수를 호출**하고,

OAuth2에서는 **등록한 URL을 호출**합니다.

### 포인트2 — OAuth2도 Callback이다

```text
내 서비스
    │ 로그인 요청
    ▼
Google
    │ 인증 완료
    ▼
GET /login/oauth2/code/google?code=...
    ▼
내 서비스
```

Google이 인증을 마친 뒤 **약속한 Callback URL(Redirect URI)** 로 다시 돌아오는 것도 Callback입니다.

작업이 끝난 뒤 **다시(Call Back) 나를 호출**하기 때문입니다.

<br>

Callback은 **함수 호출일 수도 있고, HTTP 요청일 수도 있습니다.**

> `핵심` — Callback은 "내가 시작한 작업이 끝났을 때 다시 알려주는 것"이다.
